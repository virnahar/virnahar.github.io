TDS mismatch: how to fix before filing

A mismatch does not always mean a major problem, but it should be resolved before final submission whenever possible.

Common mismatch causes

  • Salary and non-salary TDS mixed in wrong buckets.
  • Timing differences between deduction and reporting.
  • Incorrect source mapping when preparing return values.
  • Duplicate or missing entries in manual worksheets.

Step-by-step fix flow

  1. List mismatch entries by source and amount.
  2. Validate salary TDS separately from other income TDS.
  3. Cross-check with your source statements and filing draft.
  4. Adjust return values or start correction flow if already filed.
Important: Do not force values just to make totals match visually. Keep evidence-backed figures and document assumptions.
