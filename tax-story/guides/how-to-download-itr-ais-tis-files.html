How to download ITR, AIS, and TIS files

Use this quick checklist to collect the right files before analysis.

1) Download ITR JSON/XML

On incometax.gov.in, open filed returns for the target assessment year and download the available return file (JSON, or legacy XML for older years).

2) Download AIS JSON

Go to the AIS service area and export AIS JSON for the year. This is usually the most useful machine-readable source for reconciliation.

3) Download TIS PDF (optional)

If you need the summary PDF, download TIS PDF from AIS. Keep your password handy for extraction where required.

4) Upload in Tax Story

Open Tax Story, upload all files together, then review year alignment, mismatch cards, and regime comparison.

Tip: Keep consistent naming by FY and avoid duplicate files for the same FY and source when possible.
